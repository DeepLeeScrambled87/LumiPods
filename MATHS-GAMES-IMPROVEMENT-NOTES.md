# Maths Games Improvement Notes

These are the highest-value improvements identified from the current maths games implementation.

## Strongest next improvements

1. Add clearer game modes
- `Practice` for calm repetition
- `Beat the Clock` for speed
- `Boss Round` for mixed retrieval
- `Build It` for applying the idea in a tiny scenario

2. Give better feedback after each answer
- Show why the answer works, not just whether it is correct
- Surface pattern tips like repeated addition, factor families, and conversion shortcuts
- Add “show me another example” as a one-click helper

3. Make progression feel more alive
- Track streaks per game
- Unlock themed challenge cards after a level-up
- Show “next mastery target” instead of only best time and level

4. Tie maths to real tools and projects
- For conversions: quick data tables, graphing, and spreadsheet mini tasks
- For percentages: discounts, stats, and simple data stories
- For fractions: recipe scaling, measurements, and pie-chart visuals
- For multiplication: arrays, area models, and pattern machines

5. Vary the interaction patterns
- Matching
- Drag to sort
- Fill-the-gap
- Number line placement
- Build-the-model
- Explain-the-pattern

6. Use stronger visual models
- Fraction bars and circles
- Percent grids
- Number lines
- Array models for multiplication
- Roman numeral symbol builders

## Best order to implement

1. Add post-answer explanation cards
2. Add multiple game modes per concept
3. Add stronger visual manipulatives
4. Add tiny project/tool-use extensions
5. Add streaks, boss rounds, and unlockables

## Key principle

The games should not only test recall. They should help learners *see the pattern*, *use the tool*, and *apply the maths* in a playful way.
